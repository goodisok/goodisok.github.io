第五篇：复变函数在空气动力学中的应用——复势与翼型绕流

一、复势——统一描述二维势流

1.1 概念引入

在第二篇中我们学过：任意解析函数 f(z)=ϕ+iψf(z) = \phi + i\psif(z)=ϕ+iψ 的实部和虚部都是调和函数——即满足拉普拉斯方程。

在空气动力学中：

  • ϕ\phiϕ = 速度势
  • ψ\psiψ = 流函数
  • f(z)=ϕ+iψf(z) = \phi + i\psif(z)=ϕ+iψ 称为**复势**（Complex Potential）

所以：每一个解析函数对应一种可能的二维不可压缩无旋流动。

1.2 从复势得到流场信息

速度场从复势的导数获得——复速度：

dfdz=∂ϕ∂x+i∂ψ∂x=u−iv \frac{df}{dz} = \frac{\partial\phi}{\partial x} + i\frac{\partial\psi}{\partial x} = u - iv dzdf​=∂x∂ϕ​+i∂x∂ψ​=u−iv

模长：

∣dfdz∣=∣V∣ \left|\frac{df}{dz}\right| = |\mathbf{V}| ​dzdf​​=∣V∣

二、基本流动解的复势形式

2.1 均匀流

复势：

f(z)=V∞z=V∞(x+iy) f(z) = V_\infty z = V_\infty(x + iy) f(z)=V∞​z=V∞​(x+iy)

实部和虚部：

ϕ=V∞x,ψ=V∞y \phi = V_\infty x, \quad \psi = V_\infty y ϕ=V∞​x,ψ=V∞​y

→ 与第二讲 Anderson 笔记中的 ϕ=V∞x\phi = V_\infty xϕ=V∞​x 一致。

2.2 源流与汇流

复势：

f(z)=Λ2πln⁡z f(z) = \frac{\Lambda}{2\pi}\ln z f(z)=2πΛ​lnz

记 z=reiθz = re^{i\theta}z=reiθ，ln⁡z=ln⁡r+iθ\ln z = \ln r + i\thetalnz=lnr+iθ：

f(z)=Λ2π(ln⁡r+iθ) f(z) = \frac{\Lambda}{2\pi}(\ln r + i\theta) f(z)=2πΛ​(lnr+iθ)

所以：

ϕ=Λ2πln⁡r,ψ=Λ2πθ \phi = \frac{\Lambda}{2\pi}\ln r, \quad \psi = \frac{\Lambda}{2\pi}\theta ϕ=2πΛ​lnr,ψ=2πΛ​θ

→ 与 Anderson 的 ϕ=(Λ/2π)ln⁡r\phi = (\Lambda/2\pi)\ln rϕ=(Λ/2π)lnr 完全一致。

2.3 偶极子

复势：

f(z)=μ2πz f(z) = \frac{\mu}{2\pi z} f(z)=2πzμ​

写成极坐标 z=reiθz = re^{i\theta}z=reiθ，1/z=e−iθ/r=(cos⁡θ−isin⁡θ)/r1/z = e^{-i\theta}/r = (\cos\theta - i\sin\theta)/r1/z=e−iθ/r=(cosθ−isinθ)/r：

f(z)=μ2πr(cos⁡θ−isin⁡θ) f(z) = \frac{\mu}{2\pi r}(\cos\theta - i\sin\theta) f(z)=2πrμ​(cosθ−isinθ) ϕ=μ2πcos⁡θr,ψ=−μ2πsin⁡θr \phi = \frac{\mu}{2\pi}\frac{\cos\theta}{r}, \quad \psi = -\frac{\mu}{2\pi}\frac{\sin\theta}{r} ϕ=2πμ​rcosθ​,ψ=−2πμ​rsinθ​

→ 与 Anderson 的 ϕ=μcos⁡θ/(2πr)\phi = \mu\cos\theta/(2\pi r)ϕ=μcosθ/(2πr) 一致。

2.4 点涡

复势：

f(z)=Γ2πiln⁡z=−iΓ2πln⁡z f(z) = \frac{\Gamma}{2\pi i}\ln z = -i\frac{\Gamma}{2\pi}\ln z f(z)=2πiΓ​lnz=−i2πΓ​lnz f(z)=Γ2π(θ−iln⁡r) f(z) = \frac{\Gamma}{2\pi}(\theta - i\ln r) f(z)=2πΓ​(θ−ilnr)

所以：

ϕ=Γ2πθ,ψ=−Γ2πln⁡r \phi = \frac{\Gamma}{2\pi}\theta, \quad \psi = -\frac{\Gamma}{2\pi}\ln r ϕ=2πΓ​θ,ψ=−2πΓ​lnr

注意复势的简洁性：四种基本解在实变写法中各需要分别推导 ϕ\phiϕ 和 ψ\psiψ 两个函数。而复变写法只需要一个函数 f(z)f(z)f(z)。

基本解 实变形式的两个函数 复变写法的单个函数
均匀流 ϕ=V∞x\phi = V_\infty xϕ=V∞​x, ψ=V∞y\psi = V_\infty yψ=V∞​y f(z)=V∞zf(z) = V_\infty zf(z)=V∞​z
源 ϕ=Λ2πln⁡r\phi = \frac{\Lambda}{2\pi}\ln rϕ=2πΛ​lnr, ψ=Λ2πθ\psi = \frac{\Lambda}{2\pi}\thetaψ=2πΛ​θ f(z)=Λ2πln⁡zf(z) = \frac{\Lambda}{2\pi}\ln zf(z)=2πΛ​lnz

三、圆柱绕流的复势——升力的诞生

3.1 无环量圆柱绕流

叠加均匀流 + 偶极子（μ=2πV∞R2\mu = 2\pi V_\infty R^2μ=2πV∞​R2）：

f(z)=V∞z+V∞R2z f(z) = V_\infty z + \frac{V_\infty R^2}{z} f(z)=V∞​z+zV∞​R2​

在圆柱表面 z=Reiθz = Re^{i\theta}z=Reiθ 上验证：

f(Reiθ)=V∞Reiθ+V∞Re−iθ=2V∞Rcos⁡θ f(Re^{i\theta}) = V_\infty Re^{i\theta} + V_\infty Re^{-i\theta} = 2V_\infty R\cos\theta f(Reiθ)=V∞​Reiθ+V∞​Re−iθ=2V∞​Rcosθ

这是一个实数→意味着 ψ=0\psi = 0ψ=0→圆柱表面是流线 ✓

3.2 带环量的圆柱绕流

再加一个点涡：

f(z)=V∞z+V∞R2z+Γ2πiln⁡z f(z) = V_\infty z + \frac{V_\infty R^2}{z} + \frac{\Gamma}{2\pi i}\ln z f(z)=V∞​z+zV∞​R2​+2πiΓ​lnz

速度场由 df/dzdf/dzdf/dz 得到：

dfdz=V∞−V∞R2z2+Γ2πiz \frac{df}{dz} = V_\infty - \frac{V_\infty R^2}{z^2} + \frac{\Gamma}{2\pi i z} dzdf​=V∞​−z2V∞​R2​+2πizΓ​

在圆柱表面 z=Reiθz = Re^{i\theta}z=Reiθ：

dfdz=V∞(1−e−2iθ)+Γ2πiRe−iθ \frac{df}{dz} = V_\infty(1 - e^{-2i\theta}) + \frac{\Gamma}{2\pi i R}e^{-i\theta} dzdf​=V∞​(1−e−2iθ)+2πiRΓ​e−iθ

模长为：

∣V∣=∣−2iV∞e−iθsin⁡θ+Γ2πRe−iθ∣=∣2V∞sin⁡θ−Γ2πR∣ |V| = \left|-2iV_\infty e^{-i\theta}\sin\theta + \frac{\Gamma}{2\pi R} e^{-i\theta}\right| = \left|2V_\infty\sin\theta - \frac{\Gamma}{2\pi R}\right| ∣V∣=​−2iV∞​e−iθsinθ+2πRΓ​e−iθ​=​2V∞​sinθ−2πRΓ​​

这与 Anderson 的 Vθ=−2V∞sin⁡θ−Γ/(2πR)V_\theta = -2V_\infty\sin\theta - \Gamma/(2\pi R)Vθ​=−2V∞​sinθ−Γ/(2πR) 一致。

3.3 关于环量的积分为升力

使用留数定理计算作用在圆柱上的力：

Fx−iFy=iρ2∮C(dfdz)2dz F_x - iF_y = \frac{i\rho}{2}\oint_C \left(\frac{df}{dz}\right)^2 dz Fx​−iFy​=2iρ​∮C​(dzdf​)2dz

这是布拉休斯定理（Blasius Theorem）——用复变方法计算物体所受的流体力。

代入 df/dzdf/dzdf/dz 并计算留数：

Fx−iFy=iρV∞Γ F_x - iF_y = i\rho V_\infty \Gamma Fx​−iFy​=iρV∞​Γ

所以：

  • 阻力 Fx=0F_x = 0Fx​=0（达朗贝尔佯谬）
  • 升力 Fy=ρV∞ΓF_y = \rho V_\infty \GammaFy​=ρV∞​Γ（库塔-儒科夫斯基定理）

四、儒科夫斯基翼型的完全解

4.1 映射步骤

  1. 在 ζ\zetaζ 平面上取一个圆心在 (−μ,0)(-\mu, 0)(−μ,0)，半径 R=c+δR = c + \deltaR=c+δ 的圆
  2. 外加均匀来流（迎角 α\alphaα）和点涡（环量 Γ\GammaΓ）
  3. 用儒科夫斯基变换 z=ζ+c2/ζz = \zeta + c^2/\zetaz=ζ+c2/ζ 映射到物理平面

4.2 复势的完全表达式

在 ζ\zetaζ 平面上：

F(ζ)=V∞[e−iαζ+R2eiαζ−ζ0]+Γ2πiln⁡(ζ−ζ0) F(\zeta) = V_\infty\left[e^{-i\alpha}\zeta + \frac{R^2 e^{i\alpha}}{\zeta - \zeta_0}\right] + \frac{\Gamma}{2\pi i}\ln(\zeta - \zeta_0) F(ζ)=V∞​[e−iαζ+ζ−ζ0​R2eiα​]+2πiΓ​ln(ζ−ζ0​)

其中 ζ0\zeta_0ζ0​ 是圆心位置。

在物理平面 zzz 上：

f(z)=F(ζ(z)) f(z) = F(\zeta(z)) f(z)=F(ζ(z))

其中 ζ(z)\zeta(z)ζ(z) 是儒科夫斯基变换的逆变换。

4.3 升力系数

由库塔条件确定环量后：

Cl=2π(1+δc)sin⁡(α+β)+2πsin⁡β C_l = 2\pi\left(1 + \frac{\delta}{c}\right)\sin(\alpha + \beta) + 2\pi\sin\beta Cl​=2π(1+cδ​)sin(α+β)+2πsinβ

对于薄翼型（δ≪c\delta \ll cδ≪c，β\betaβ 小），还原为：

Cl=2π(α−αL=0) C_l = 2\pi(\alpha - \alpha_{L=0}) Cl​=2π(α−αL=0​)

这正是薄翼理论的核心结果。
