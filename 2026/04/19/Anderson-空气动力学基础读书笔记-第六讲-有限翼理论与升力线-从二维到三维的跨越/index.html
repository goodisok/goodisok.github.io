Anderson《空气动力学基础》读书笔记（六）：有限翼理论与升力线——从二维到三维的跨越

发表于2026-04-19|更新于2026-04-19|空气动力学基础
|浏览量:

本文是 John D. Anderson《Fundamentals of Aerodynamics》（第六版）读书笔记系列的第六篇，覆盖教材第7章《有限翼理论》（Prandtl’s Lifting-Line Theory）。这是Anderson全书中最具工程实用价值的一章之一。

第六篇：有限翼理论与升力线——从二维到三维的跨越

一、为什么二维理论不够？

1.1 概念引入

薄翼理论给出了一个完美的二维翼型：Cl=2παC_l = 2\pi\alphaCl​=2πα，CdC_dCd​ 很小，唯一的力是升力。但现实中的机翼是有限展长的——有两个端部（翼尖），空气可以从下表面通过翼尖流到上表面。

这个”翼端泄漏”效应会带来三个重要后果：

  1. 升力降低：在翼尖处升力降为零
  2. 诱导阻力出现：额外的阻力，在二维中不存在
  3. 翼尖涡形成：真实机翼后面拖出两条集中的涡

Anderson 将普朗特（Prandtl）在 1918-1919 年左右提出的升力线理论描述为”空气动力学史上最优雅的理论之一”。

1.2 物理机制

升力沿着展向分布——翼根处大，翼尖处小。展向升力梯度意味着环量沿着展向变化。

根据亥姆霍兹涡量定理，涡量不能在流体内部终止。所以：

  • 升力变化的地方 → 涡面从机翼后缘拖出
  • 在翼尖处 → 集中卷成翼尖涡（wingtip vortices）

这些拖出的自由涡在机翼处诱导了一个向下的速度（下洗，downwash），改变了当地迎角，从而产生了诱导阻力。

二、升力线理论的数学框架

2.1 核心假设

普朗特升力线理论基于以下假设：

  1. 机翼用一根在展向位置的升力线（lifting line）替代，通常是 1/4 弦线
  2. 环量 Γ(y)\Gamma(y)Γ(y) 沿展向分布，在翼尖处 Γ(±b/2)=0\Gamma(\pm b/2)=0Γ(±b/2)=0
  3. 后缘拖出的自由涡形成一个涡面（vortex sheet）
  4. 流动是不可压缩的
  5. 机翼是平面直机翼（无后掠，无扭转）

2.2 下洗与诱导迎角

下洗速度 w(y)w(y)w(y) 由展向涡量分布在 Biot-Savart 定律下产生：

w(y0)=14π∫−b/2b/2dΓ/dyy0−y dy w(y_0) = \frac{1}{4\pi}\int_{-b/2}^{b/2} \frac{d\Gamma/dy}{y_0 - y}\,dy w(y0​)=4π1​∫−b/2b/2​y0​−ydΓ/dy​dy

这个速度使来流迎角减小了 αi\alpha_iαi​：

αi(y)=−w(y)V∞ \alpha_i(y) = -\frac{w(y)}{V_\infty} αi​(y)=−V∞​w(y)​

诱导迎角（Induced Angle of Attack） 使得实际迎角 αeff\alpha_{\text{eff}}αeff​ 小于几何迎角 α\alphaα：

αeff(y)=α(y)−αi(y) \alpha_{\text{eff}}(y) = \alpha(y) - \alpha_i(y) αeff​(y)=α(y)−αi​(y)

二维翼型的升力由有效迎角决定，而不是几何迎角。

2.3 升力线积分方程

升力线理论的核心——一个积分方程将环量分布 Γ(y)\Gamma(y)Γ(y) 与机翼的几何参数连接起来：

Γ(y)=12c(y)V∞a0(y)[α(y)−αL=0(y)−14πV∞∫−b/2b/2dΓ/dηy−η dη] \Gamma(y) = \frac{1}{2} c(y) V_\infty a_0(y) \left[\alpha(y) - \alpha_{L=0}(y) - \frac{1}{4\pi V_\infty}\int_{-b/2}^{b/2} \frac{d\Gamma/d\eta}{y-\eta}\,d\eta\right] Γ(y)=21​c(y)V∞​a0​(y)[α(y)−αL=0​(y)−4πV∞​1​∫−b/2b/2​y−ηdΓ/dη​dη]

这个方程是普朗特的升力线方程。Mathematically formidable, physically beautiful.

2.4 傅里叶级数解

Glauert 引入了一个变换——将展向位置用角度表示：

y=−b2cos⁡θ,0≤θ≤π y = -\frac{b}{2}\cos\theta, \quad 0 \le \theta \le \pi y=−2b​cosθ,0≤θ≤π

Γ(θ)=2bV∞∑n=1∞Ansin⁡(nθ) \Gamma(\theta) = 2b V_\infty \sum_{n=1}^\infty A_n \sin(n\theta) Γ(θ)=2bV∞​n=1∑∞​An​sin(nθ)

三、椭圆翼——最优雅的解析解

3.1 椭圆环量分布

如果机翼是椭圆环量分布——即 Γ(y)=Γ01−(2y/b)2\Gamma(y) = \Gamma_0 \sqrt{1 - (2y/b)^2}Γ(y)=Γ0​1−(2y/b)2​，或者相应地，弦长 c(y)c(y)c(y) 沿展向呈椭圆分布——那么积分方程有精确的解析解。

椭圆翼的环量分布：

Γ(θ)=2bV∞A1sin⁡θ \Gamma(\theta) = 2b V_\infty A_1 \sin\theta Γ(θ)=2bV∞​A1​sinθ

只有第一项傅里叶系数非零。这意味着：

  • 下洗速度 www 沿展向是常数
  • 诱导迎角 αi\alpha_iαi​ 沿展向是常数

3.2 椭圆翼的关键公式

对于椭圆翼：

CL=a01+a0/(πAR)α C_L = \frac{a_0}{1 + a_0/(\pi AR)}\alpha CL​=1+a0​/(πAR)a0​​α

其中 a0a_0a0​ 是二维升力线斜率（2π2\pi2π 每弧度），AR=b2/SAR = b^2/SAR=b2/S 是展弦比。

对于 a0=2πa_0 = 2\pia0​=2π：

CL=2πARAR+2α C_L = \frac{2\pi AR}{AR + 2}\alpha CL​=AR+22πAR​α

诱导阻力系数：

CD,i=CL2πAR \boxed{C_{D,i} = \frac{C_L^2}{\pi AR}} CD,i​=πARCL2​​​

这是升力线理论最著名的工程结果：诱导阻力与升力的平方成正比，与展弦比成反比。

4.3 工程启示

展弦比大的机翼（AR>10AR > 10AR>10）：

  • 升力线斜率接近二维值
  • 诱导阻力低
  • 适合长续航飞行（滑翔机、侦察无人机）
  • 缺点：结构重、大过载时翼根弯矩大

展弦比小的机翼（AR<5AR < 5AR<5）：

  • 升力严重降级
  • 诱导阻力大
  • 适合大过载机动（战斗机、竞速机）
  • 结构紧凑，刚度好

四旋翼臂作为”机翼”的情况：四旋翼的机身/机臂通常展弦比为 1-3，诱导阻力效应非常显著。这也是为什么高速截击机会采用固定翼机身或者整机设计而非简单的四旋翼构型。

五、非椭圆翼和 Oswald 效率因子

5.1 非椭圆翼的修正

对于非椭圆环量分布的机翼，诱导阻力公式修正为：

其中 eee 是Oswald 效率因子（0<e≤10 < e \le 10<e≤1）。
