一、概率论的基本概念

概念理解

概率论回答一个根本问题：在不确定中，我们能知道什么？

抛一枚硬币，你不知道下一次是正面还是反面——但你知道抛1000次，大约500次正面。概率论就是把这种”大约”精确化的语言。

核心定义

样本空间 Ω\OmegaΩ：随机试验所有可能结果的集合。

事件：Ω\OmegaΩ 的子集。事件 AAA 发生 = 试验结果落在 AAA 中。

概率的公理化定义（柯尔莫哥洛夫，1933）：

  1. 非负性：P(A)≥0P(A) \ge 0P(A)≥0
  2. 规范性：P(Ω)=1P(\Omega) = 1P(Ω)=1
  3. 可列可加性：互不相容事件的概率 = 各自概率之和

关键性质

条件概率：已知 BBB 发生的条件下，AAA 发生的概率

P(A∣B)=P(AB)P(B)P(A|B) = \frac{P(AB)}{P(B)}P(A∣B)=P(B)P(AB)​

乘法公式：P(AB)=P(B)P(A∣B)=P(A)P(B∣A)P(AB) = P(B)P(A|B) = P(A)P(B|A)P(AB)=P(B)P(A∣B)=P(A)P(B∣A)

全概率公式：若 B1,…,BnB_1,\ldots,B_nB1​,…,Bn​ 构成完备事件组（互不相容且并集为 Ω\OmegaΩ），则

P(A)=∑i=1nP(Bi)P(A∣Bi)P(A) = \sum_{i=1}^{n} P(B_i)P(A|B_i)P(A)=i=1∑n​P(Bi​)P(A∣Bi​)

全概率公式的本质：把复杂事件拆成”路径”——所有路径概率之和就是总概率。

贝叶斯公式（本文的核心——从结果反推原因）：

P(Bi∣A)=P(Bi)P(A∣Bi)∑j=1nP(Bj)P(A∣Bj)P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\sum_{j=1}^{n}P(B_j)P(A|B_j)}P(Bi​∣A)=∑j=1n​P(Bj​)P(A∣Bj​)P(Bi​)P(A∣Bi​)​
  • P(Bi)P(B_i)P(Bi​) = 先验概率（在看到数据前的信念）
  • P(A∣Bi)P(A|B_i)P(A∣Bi​) = 似然（在假设 BiB_iBi​ 下观测到 AAA 的可能性）
  • P(Bi∣A)P(B_i|A)P(Bi​∣A) = 后验概率（结合数据后的更新信念）

独立性：P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B)。直观：知道 AAA 发生与否，不影响 BBB 的概率。

手算例子

箱子 A 有 3 红球 2 白球，箱子 B 有 1 红球 4 白球。随机选一个箱子，抽到红球的概率？

全概率：P(红)=P(A)P(红∣A)+P(B)P(红∣B)=12⋅35+12⋅15=25P(\text{红}) = P(A)P(\text{红}|A) + P(B)P(\text{红}|B) = \frac{1}{2} \cdot \frac{3}{5} + \frac{1}{2} \cdot \frac{1}{5} = \frac{2}{5}P(红)=P(A)P(红∣A)+P(B)P(红∣B)=21​⋅53​+21​⋅51​=52​

假如抽到了红球，这球来自箱子 A 的概率？

贝叶斯：P(A∣红)=P(A)P(红∣A)P(红)=12⋅3525=34P(A|\text{红}) = \dfrac{P(A)P(\text{红}|A)}{P(\text{红})} = \dfrac{\frac{1}{2}\cdot\frac{3}{5}}{\frac{2}{5}} = \frac{3}{4}P(A∣红)=P(红)P(A)P(红∣A)​=52​21​⋅53​​=43​

工程应用

  • 卡尔曼滤波：先验（运动模型）→ 似然（传感器观测）→ 后验（状态估计），就是贝叶斯递归
  • 垃圾邮件分类：P(垃圾∣包含"赚钱")=P(垃圾)P(包含"赚钱"∣垃圾)P(包含"赚钱")P(\text{垃圾}|\text{包含"赚钱"}) = \frac{P(\text{垃圾})P(\text{包含"赚钱"}|\text{垃圾})}{P(\text{包含"赚钱"})}P(垃圾∣包含"赚钱")=P(包含"赚钱")P(垃圾)P(包含"赚钱"∣垃圾)​

二、随机变量及其分布

概念理解

随机变量就是把随机结果映射成数字。它有两个核心角色：

  • 分布函数（CDF）F(x)=P(X≤x)F(x) = P(X \le x)F(x)=P(X≤x)：描述”X 不超过 x”的概率
  • 概率密度（PDF）f(x)f(x)f(x)：连续型随机变量的”概率浓度”——不是某点的概率，而是宽度趋近零时的比例

核心定义

类型 概率质量/密度 分布函数
离散型 P(X=xk)=pkP(X = x_k) = p_kP(X=xk​)=pk​ F(x)=∑xk≤xpkF(x) = \sum_{x_k \le x} p_kF(x)=∑xk​≤x​pk​
连续型 密度 f(x)≥0f(x) \ge 0f(x)≥0，∫−∞+∞f(x)dx=1\int_{-\infty}^{+\infty}f(x)dx = 1∫−∞+∞​f(x)dx=1 F(x)=∫−∞xf(t)dtF(x) = \int_{-\infty}^{x} f(t)dtF(x)=∫−∞x​f(t)dt

重要关系：f(x)=F′(x)f(x) = F'(x)f(x)=F′(x)（几乎处处）。CDF 的斜率就是 PDF。

三、多维随机变量及其分布

概念理解

现实中很少有独立的量。无人机的位置和速度高度耦合——位置的变化依赖于速度，速度的变化依赖于加速度。多维随机变量描述的就是这种联合不确定性。

核心定义

联合分布函数：F(x,y)=P(X≤x,Y≤y)F(x, y) = P(X \le x, Y \le y)F(x,y)=P(X≤x,Y≤y)

联合密度：f(x,y)f(x, y)f(x,y)，满足

F(x,y)=∫−∞x∫−∞yf(u,v) du dvF(x, y) = \int_{-\infty}^{x} \int_{-\infty}^{y} f(u, v) \, du \, dvF(x,y)=∫−∞x​∫−∞y​f(u,v)dudv P((X,Y)∈D)=∬Df(x,y) dx dyP((X,Y) \in D) = \iint_D f(x, y) \, dx \, dyP((X,Y)∈D)=∬D​f(x,y)dxdy

边缘分布：单独看 XXX 或 YYY

fX(x)=∫−∞+∞f(x,y) dyf_X(x) = \int_{-\infty}^{+\infty} f(x, y) \, dyfX​(x)=∫−∞+∞​f(x,y)dy fY(y)=∫−∞+∞f(x,y) dxf_Y(y) = \int_{-\infty}^{+\infty} f(x, y) \, dxfY​(y)=∫−∞+∞​f(x,y)dx

条件分布：已知 Y=yY=yY=y 时 XXX 的分布

fX∣Y(x∣y)=f(x,y)fY(y)f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}fX∣Y​(x∣y)=fY​(y)f(x,y)​

关键性质

协方差：度量两个变量”同方向变化”的程度

Cov(X,Y)=E[(X−EX)(Y−EY)]=E(XY)−E(X)E(Y)\text{Cov}(X, Y) = E[(X-EX)(Y-EY)] = E(XY) - E(X)E(Y)Cov(X,Y)=E[(X−EX)(Y−EY)]=E(XY)−E(X)E(Y)
  • 正 → X 大时 Y 也偏大
  • 负 → X 大时 Y 偏小
  • 零 → 不相关（但不一定独立！）

四、随机变量的数字特征

概念理解

分布（PDF/CDF）是完全信息，但通常太复杂。数字特征是用一两个数概括整体分布——就像用平均分概括全班成绩。

核心定义

数学期望：加权平均（”概率加权”）

离散：E(X)=∑kxkpkE(X) = \sum_k x_k p_kE(X)=∑k​xk​pk​

连续：E(X)=∫−∞+∞xf(x)dxE(X) = \int_{-\infty}^{+\infty} x f(x) dxE(X)=∫−∞+∞​xf(x)dx

方差：偏离均值的”平均平方”

D(X)=E[(X−EX)2]=E(X2)−(EX)2D(X) = E[(X-EX)^2] = E(X^2) - (EX)^2D(X)=E[(X−EX)2]=E(X2)−(EX)2

标准差 σ=D(X)\sigma = \sqrt{D(X)}σ=D(X)​，和原始量纲相同。

切比雪夫不等式：不管分布长什么样，”偏离的极限”是确定的

P(∣X−μ∣≥ε)≤σ2ε2P(|X-\mu| \ge \varepsilon) \le \frac{\sigma^2}{\varepsilon^2}P(∣X−μ∣≥ε)≤ε2σ2​

这是大数定律的数学基础。

矩：一般化的数字特征

  • kkk 阶原点矩：E(Xk)E(X^k)E(Xk)（期望 = 一阶原点矩）
  • kkk 阶中心矩：E[(X−EX)k]E[(X-EX)^k]E[(X−EX)k]（方差 = 二阶中心矩）
  • 偏度 = 三阶中心矩 / σ3\sigma^3σ3：分布的不对称性
  • 峰度 = 四阶中心矩 / σ4−3\sigma^4 - 3σ4−3：尾部厚度

关键性质

期望的线性性：E(aX+bY)=aE(X)+bE(Y)E(aX + bY) = aE(X) + bE(Y)E(aX+bY)=aE(X)+bE(Y)（永远成立，不管是否独立）

独立时：E(XY)=E(X)E(Y)E(XY) = E(X)E(Y)E(XY)=E(X)E(Y)，D(X+Y)=D(X)+D(Y)D(X+Y) = D(X) + D(Y)D(X+Y)=D(X)+D(Y)

卷积公式：独立随机变量和的密度 = 各自密度的卷积

fX+Y(z)=∫−∞+∞fX(x)fY(z−x) dxf_{X+Y}(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) \, dxfX+Y​(z)=∫−∞+∞​fX​(x)fY​(z−x)dx

五、大数定律与中心极限定理

概念理解

这两条定理是整个统计推断的基石：

  • 大数定律：样本均值收敛到总体均值（你猜得够多，就会猜对）
  • 中心极限定理：独立随机变量之和近似正态分布（不管原始分布长什么样）

核心定义

辛钦大数定律（弱大数）：X1,…,XnX_1,\ldots,X_nX1​,…,Xn​ 独立同分布且期望 μ\muμ 存在，则

Xˉn=1n∑i=1nXi⟶Pμ\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \stackrel{P}{\longrightarrow} \muXˉn​=n1​i=1∑n​Xi​⟶P​μ

符号 ⟶P\stackrel{P}{\longrightarrow}⟶P​ 表示”依概率收敛”——随着 nnn 增大，偏离的概率趋于零。

Lindeberg-Lévy 中心极限定理：独立同分布的 XiX_iXi​，期望 μ\muμ，方差 σ2>0\sigma^2 > 0σ2>0，则

∑i=1nXi−nμnσ⟶dN(0,1)\frac{\sum_{i=1}^n X_i - n\mu}{\sqrt{n}\sigma} \stackrel{d}{\longrightarrow} N(0, 1)n​σ∑i=1n​Xi​−nμ​⟶d​N(0,1)

等价形式：

Xˉn∼近似N(μ,σ2n)\bar{X}_n \stackrel{\text{近似}}{\sim} N\left(\mu, \frac{\sigma^2}{n}\right)Xˉn​∼近似N(μ,nσ2​)

关键洞察：样本均值的方差是总体方差的 1/n1/n1/n——这就是为什么样本量翻倍，精度提高 2\sqrt{2}2​ 倍而非 2 倍。

手算例子

抛硬币 60 次，正面概率 0.5。求正面次数在 25~35 之间的概率。

X∼B(60,0.5)X \sim B(60, 0.5)X∼B(60,0.5)，用正态近似（中心极限）： μ=np=30\mu = np = 30μ=np=30，σ2=np(1−p)=15\sigma^2 = np(1-p) = 15σ2=np(1−p)=15 P(25≤X≤35)≈Φ(35.5−3015)−Φ(24.5−3015)P(25 \le X \le 35) \approx \Phi\left(\frac{35.5 - 30}{\sqrt{15}}\right) - \Phi\left(\frac{24.5 - 30}{\sqrt{15}}\right)P(25≤X≤35)≈Φ(15​35.5−30​)−Φ(15​24.5−30​) ≈Φ(1.42)−Φ(−1.42)≈0.922−0.078=0.844\approx \Phi(1.42) - \Phi(-1.42) \approx 0.922 - 0.078 = 0.844≈Φ(1.42)−Φ(−1.42)≈0.922−0.078=0.844

工程应用

  • 蒙特卡洛仿真：nnn 次试验，精度 ∝1/n\propto 1/\sqrt{n}∝1/n​——这就是为什么从100次到10000次只提高10倍精度
  • A/B 测试：均值差异的显著性检验，底层是中心极限定理
  • 传感器融合：多个独立传感器的平均，精度随传感器数量增加而提高

六、样本及抽样分布

概念理解

前面五章是概率论——已知分布，预测结果。从第六章起是数理统计——已知结果，推断分布。概率论和数理统计互为逆问题：

概率论：分布 → 数据

数理统计：数据 → 分布

核心定义

总体：研究对象的全体。样本：从总体中抽出的部分个体。

统计量：样本的函数（不含未知参数），比如样本均值 Xˉ\bar{X}Xˉ、样本方差 S2S^2S2。

三大抽样分布——假设检验和置信区间的计算都依赖它们：

（1）χ2\chi^2χ2 分布：X1,…,Xn∼N(0,1)X_1,\ldots,X_n \sim N(0,1)X1​,…,Xn​∼N(0,1) 独立，则

∑i=1nXi2∼χ2(n)\sum_{i=1}^n X_i^2 \sim \chi^2(n)i=1∑n​Xi2​∼χ2(n) E[χ2(n)]=n,D[χ2(n)]=2nE[\chi^2(n)] = n, \quad D[\chi^2(n)] = 2nE[χ2(n)]=n,D[χ2(n)]=2n

（2）ttt 分布（学生氏分布）：X∼N(0,1)X \sim N(0,1)X∼N(0,1)，Y∼χ2(n)Y \sim \chi^2(n)Y∼χ2(n) 独立，则

T=XY/n∼t(n)T = \frac{X}{\sqrt{Y/n}} \sim t(n)T=Y/n​X​∼t(n) nnn 增大时 ttt 分布趋近标准正态——n≥30n \ge 30n≥30 时可用正态近似。

（3）FFF 分布：X∼χ2(n1)X \sim \chi^2(n_1)X∼χ2(n1​)，Y∼χ2(n2)Y \sim \chi^2(n_2)Y∼χ2(n2​) 独立，则

F=X/n1Y/n2∼F(n1,n2)F = \frac{X/n_1}{Y/n_2} \sim F(n_1, n_2)F=Y/n2​X/n1​​∼F(n1​,n2​)

用于比较两个方差是否相等（方差分析）。

关键事实——正态总体下的结论

设 X1,…,XnX_1,\ldots,X_nX1​,…,Xn​ 是来自 N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2) 的样本，则：

  1. Xˉ∼N(μ,σ2n)\bar{X} \sim N(\mu, \frac{\sigma^2}{n})Xˉ∼N(μ,nσ2​)
  2. (n−1)S2σ2∼χ2(n−1)\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)σ2(n−1)S2​∼χ2(n−1)
  3. Xˉ\bar{X}Xˉ 与 S2S^2S2 独立（正态分布独有的神奇性质）
  4. Xˉ−μS/n∼t(n−1)\frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)S/n​Xˉ−μ​∼t(n−1)（σ\sigmaσ 未知时用 SSS 代替 → t 分布）

工程应用

  • t 检验 → 小样本均值比较（< 30 样本）
  • F 检验 → 方差分析（多个总体均值比较）、回归显著性
  • χ2\chi^2χ2 检验 → 拟合优度检验、列联表的独立性检验

七、参数估计

概念理解

已知总体分布的类型（比如知道是正态），但参数 θ\thetaθ 未知。怎么用样本估计 θ\thetaθ？

两个流派：

  • 点估计：给一个数（比如 μ^=xˉ\hat{\mu} = \bar{x}μ^​=xˉ）
  • 区间估计：给一个区间和置信度（比如 μ\muμ 的 95% 置信区间）

核心方法

矩估计法：用样本矩替代总体矩，解出参数。

比如 X∼U(0,θ)X \sim U(0, \theta)X∼U(0,θ)，E(X)=θ/2E(X) = \theta/2E(X)=θ/2 → θ^=2Xˉ\hat{\theta} = 2\bar{X}θ^=2Xˉ。

最大似然估计（MLE）——整个统计学的核心方法：

L(θ)=∏i=1nf(xi;θ)L(\theta) = \prod_{i=1}^n f(x_i; \theta)L(θ)=i=1∏n​f(xi​;θ) θ^MLE=arg⁡max⁡θL(θ) \hat{\theta}_{\text{MLE}} = \arg\max_\theta L(\theta)θ^MLE​=argθmax​L(θ)

直觉：选那个”在这些参数下，观测到的数据最可能发生”的 θ\thetaθ。

评价标准

  • 无偏性：E(θ^)=θE(\hat{\theta}) = \thetaE(θ^)=θ——平均来说估计对的
  • 有效性：方差越小越好（达到克拉美-罗下界最优）
  • 相合性：样本越大估计越准（θ^n⟶Pθ\hat{\theta}_n \stackrel{P}{\longrightarrow} \thetaθ^n​⟶P​θ）

关键事实：Xˉ\bar{X}Xˉ 是 μ\muμ 的无偏估计，S2S^2S2 是 σ2\sigma^2σ2 的无偏估计（注意分母是 n−1n-1n−1 而非 nnn）。

区间估计

σ2\sigma^2σ2 已知，μ\muμ 的置信区间（1−α1-\alpha1−α 置信度）：

Xˉ±zα/2⋅σn\bar{X} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}Xˉ±zα/2​⋅n​σ​

Xˉ±tα/2(n−1)⋅Sn\bar{X} \pm t_{\alpha/2}(n-1) \cdot \frac{S}{\sqrt{n}}Xˉ±tα/2​(n−1)⋅n​S​

工程应用

  • MLE → 深度学习损失函数的设计原理（交叉熵 = 分类的负对数似然）

八、假设检验

概念理解

假设检验是反证法：先假设你想推翻的东西为真（原假设 H0H_0H0​），然后看数据有多不配合。如果数据在 H0H_0H0​ 下”太离谱”，就拒绝 H0H_0H0​。

核心定义

两类错误：

接受 H0H_0H0​ 拒绝 H0H_0H0​
H0H_0H0​ 为真 ✓ 第一类错误（α\alphaα）
H0H_0H0​ 为假 第二类错误（β\betaβ） ✓
  • α\alphaα = 显著性水平（你允许"冤枉好人"的概率上限，通常 0.05）
  • 1−β1-\beta1−β = 检验功效（"抓出坏人"的能力）
  • α\alphaα 和 β\betaβ 不能同时减小（样本量固定时），需要 trade-off

p 值：在 H0H_0H0​ 为真的假设下，观测到”当前结果或更极端”的概率。

决策规则：p < α\alphaα → 拒绝 H0H_0H0​；p ≥\ge≥ α\alphaα → 不拒绝 H0H_0H0​

注意：”不拒绝 H0H_0H0​“ ≠ “接受了 H0H_0H0​“！统计结论永远用保守措辞。

常见检验流程

1. 三步走：设 H0H_0H0​ 和 H1H_1H1​ → 选检验统计量和拒绝域 → 计算 p 值，做决策

2. 单正态总体均值的 t 检验（σ\sigmaσ 未知，最常用）：

T=Xˉ−μ0S/n∼t(n−1)T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}} \sim t(n-1)T=S/n​Xˉ−μ0​​∼t(n−1)

3. 双正态总体均值比较的 t 检验（两独立样本）：

T=Xˉ−YˉSw1n1+1n2T = \frac{\bar{X} - \bar{Y}}{S_w \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}T=Sw​n1​1​+n2​1​​Xˉ−Yˉ​

其中 Sw2S_w^2Sw2​ 是合并方差估计。

4. 方差检验（F 检验）：

F=S12S22∼F(n1−1,n2−1)F = \frac{S_1^2}{S_2^2} \sim F(n_1-1, n_2-1)F=S22​S12​​∼F(n1​−1,n2​−1)
