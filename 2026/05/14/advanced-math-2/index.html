一、向量代数与空间解析几何——从平面到空间

概念直觉

上册的函数都是 y=f(x)y=f(x)y=f(x)：一个输入、一个输出。但世界不是一维的。一架无人机的位置是 (x,y,z)(x,y,z)(x,y,z)，姿态是 (ϕ,θ,ψ)(\phi,\theta,\psi)(ϕ,θ,ψ)——六个输入。我们需要一种能在三维空间里做数学的工具，这就是向量。

向量基本运算

点积（内积）：

a⋅b=∣a∣∣b∣cos⁡θ\mathbf{a}\cdot\mathbf{b} = |\mathbf{a}||\mathbf{b}|\cos\thetaa⋅b=∣a∣∣b∣cosθ

物理意义：力 F\mathbf{F}F 沿位移 s\mathbf{s}s 做的功 W=F⋅sW = \mathbf{F}\cdot\mathbf{s}W=F⋅s。如果力和位移垂直，不做功——这就是为什么轨道速度不改变轨道高度。

叉积（外积）：

a×b=∣a∣∣b∣sin⁡θ  n\mathbf{a}\times\mathbf{b} = |\mathbf{a}||\mathbf{b}|\sin\theta\;\mathbf{n}a×b=∣a∣∣b∣sinθn

结果是垂直于两个向量的新向量，方向由右手定则确定。

物理意义：力矩 τ=r×F\boldsymbol{\tau} = \mathbf{r}\times\mathbf{F}τ=r×F，角动量 L=r×p\mathbf{L} = \mathbf{r}\times\mathbf{p}L=r×p。四旋翼的每个旋翼产生的力矩就是位置向量叉积推力向量。

混合积 (a×b)⋅c(\mathbf{a}\times\mathbf{b})\cdot\mathbf{c}(a×b)⋅c：平行六面体的体积。三个向量共面时混合积为零——判断共面的代数方法。

空间中的平面与直线

平面方程：Ax+By+Cz+D=0Ax+By+Cz+D=0Ax+By+Cz+D=0，法向量 n=(A,B,C)\mathbf{n}=(A,B,C)n=(A,B,C)。

x−x0m=y−y0n=z−z0p\frac{x-x_0}{m} = \frac{y-y_0}{n} = \frac{z-z_0}{p}mx−x0​​=ny−y0​​=pz−z0​​

方向向量 s=(m,n,p)\mathbf{s}=(m,n,p)s=(m,n,p)。

  • 两平面夹角 = 两法向量夹角，cos⁡θ=∣n1⋅n2∣∣n1∣∣n2∣\cos\theta = \frac{|\mathbf{n}_1\cdot\mathbf{n}_2|}{|\mathbf{n}_1||\mathbf{n}_2|}cosθ=∣n1​∣∣n2​∣∣n1​⋅n2​∣​

二、多元函数微分法——高维世界的导数

概念直觉

上册的导数 f′(x)f'(x)f′(x) 只有一个方向——沿着 xxx 轴。多元函数 f(x,y,z)f(x,y,z)f(x,y,z) 有无数个方向可以移动。沿每个方向的导数都不一样。怎么描述？

偏导数

对 xxx 求偏导：把 yyy 和 zzz 当作常数，只对 xxx 求导。记法：

∂f∂x,fx\frac{\partial f}{\partial x},\quad f_x∂x∂f​,fx​

几何意义：用 y=y=y= 常数的平面去截曲面 z=f(x,y)z=f(x,y)z=f(x,y)，交线是一条曲线，该曲线在 (x0,y0)(x_0,y_0)(x0​,y0​) 处的切线斜率就是 ∂f∂x\frac{\partial f}{\partial x}∂x∂f​。

全微分

一元微积分：dy=f′(x) dxdy = f'(x)\,dxdy=f′(x)dx。多元推广：

dz=∂f∂xdx+∂f∂ydydz = \frac{\partial f}{\partial x}dx + \frac{\partial f}{\partial y}dydz=∂x∂f​dx+∂y∂f​dy

这是线性近似在多元的版本：函数的变化 ≈ 各分量变化 × 对应偏导数之和。

链式法则（多元版）

一元：ddxf(g(x))=f′(g(x))g′(x)\frac{d}{dx}f(g(x)) = f'(g(x))g'(x)dxd​f(g(x))=f′(g(x))g′(x)。

多元：

ddtf(x(t),y(t))=∂f∂xdxdt+∂f∂ydydt\frac{d}{dt}f(x(t),y(t)) = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt}dtd​f(x(t),y(t))=∂x∂f​dtdx​+∂y∂f​dtdy​

这就是反向传播的数学根基。 深度学习里每一个神经元的梯度就是沿着这张链式法则的网络逐层传回去的。

方向导数与梯度——这一章的灵魂

沿单位向量 u\mathbf{u}u 方向的导数：

Duf=∇f⋅uD_{\mathbf{u}}f = \nabla f \cdot \mathbf{u}Du​f=∇f⋅u

其中 ∇f=(∂f∂x,∂f∂y)\nabla f = \left(\frac{\partial f}{\partial x},\frac{\partial f}{\partial y}\right)∇f=(∂x∂f​,∂y∂f​) 是梯度向量。

梯度的三个事实：

  1. 梯度指向函数增长最快的方向
  2. 梯度的模就是最大方向导数
  3. 梯度垂直于等值线（等高线）

多元函数的极值

一元：f′(x)=0f'(x)=0f′(x)=0 且 f′′(x)>0f''(x)>0f′′(x)>0 → 极小值。

多元需要看二阶偏导数组成的 Hessian 矩阵：

H=[fxxfxyfyxfyy]H = \begin{bmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{bmatrix}H=[fxx​fyx​​fxy​fyy​​]

判断规则（AC−B2AC-B^2AC−B2 判别法）：

AC−B2AC-B^2AC−B2 A=fxxA=f_{xx}A=fxx​ 结论
>0>0>0 >0>0>0 极小值
>0>0>0 <0<0<0 极大值
<0<0<0 任意 鞍点（不是极值）
=0=0=0 任意 无法判断（需更高阶）

拉格朗日乘数法：带约束的最优化。求 f(x,y)f(x,y)f(x,y) 在 g(x,y)=0g(x,y)=0g(x,y)=0 条件下的极值：

L(x,y,λ)=f(x,y)+λg(x,y)L(x,y,\lambda) = f(x,y) + \lambda g(x,y)L(x,y,λ)=f(x,y)+λg(x,y)

然后令 ∂L∂x=∂L∂y=∂L∂λ=0\frac{\partial L}{\partial x}=\frac{\partial L}{\partial y}=\frac{\partial L}{\partial \lambda}=0∂x∂L​=∂y∂L​=∂λ∂L​=0 联立求解。λ\lambdaλ 叫拉格朗日乘子，物理意义：约束”松一点”时目标函数的边际变化。

三、重积分——在平面和空间中求和

概念直觉

定积分是一维区域的求和（线段）。重积分是二维或三维区域的求和。

面积上的累加 → 二重积分。体积上的累加 → 三重积分。

二重积分

区域 DDD 上对 f(x,y)f(x,y)f(x,y) 积分：

∬Df(x,y) dx dy\iint_D f(x,y)\,dx\,dy∬D​f(x,y)dxdy

几何意义：曲面 z=f(x,y)z=f(x,y)z=f(x,y) 在区域 DDD 下围成的体积。

计算方法：化成两次定积分（累次积分）。先固定 xxx，对 yyy 积分（得到 xxx 截面面积），再对 xxx 积分。

交换积分次序：dx dy↔dy dxdx\,dy \leftrightarrow dy\,dxdxdy↔dydx。次序不同，难度可能天差地别——学会看积分区域的形状选次序。

极坐标变换

当积分区域是圆或环形时，直角坐标系被积分区域卡死。换到极坐标：

x=rcos⁡θ,  y=rsin⁡θ,  dx dy=r dr dθx=r\cos\theta,\; y=r\sin\theta,\; dx\,dy = r\,dr\,d\thetax=rcosθ,y=rsinθ,dxdy=rdrdθ

那个多出来的 rrr 别忘——它是坐标变换的雅可比行列式，代表面积微元从矩形变成扇形的面积补偿。

三重积分

∭Ωf(x,y,z) dx dy dz\iiint_\Omega f(x,y,z)\,dx\,dy\,dz∭Ω​f(x,y,z)dxdydz

柱坐标：x=rcos⁡θ,  y=rsin⁡θ,  z=z,  dx dy dz=r dr dθ dzx=r\cos\theta,\; y=r\sin\theta,\; z=z,\; dx\,dy\,dz = r\,dr\,d\theta\,dzx=rcosθ,y=rsinθ,z=z,dxdydz=rdrdθdz

球坐标：x=ρsin⁡ϕcos⁡θ,  y=ρsin⁡ϕsin⁡θ,  z=ρcos⁡ϕ,  dx dy dz=ρ2sin⁡ϕ dρ dϕ dθx=\rho\sin\phi\cos\theta,\; y=\rho\sin\phi\sin\theta,\; z=\rho\cos\phi,\; dx\,dy\,dz = \rho^2\sin\phi\,d\rho\,d\phi\,d\thetax=ρsinϕcosθ,y=ρsinϕsinθ,z=ρcosϕ,dxdydz=ρ2sinϕdρdϕdθ

选坐标系的铁律：区域是什么形状就用对应的坐标系。球→球坐标，柱→柱坐标。硬用直角坐标等于自虐。

四、曲线积分与曲面积分——沿着”线”和”面”积分

概念直觉

前面的积分都在”区域”上——线段、矩形、球体。但如果我要算沿着一条曲线做的功，或者穿出一个曲面的流量呢？

这就是曲线积分和曲面积分。

第一类曲线积分（对弧长）

沿着曲线 LLL 对函数 f(x,y)f(x,y)f(x,y) 积分：

∫Lf(x,y) ds\int_L f(x,y)\,ds∫L​f(x,y)ds

物理意义：质量密度为 f(x,y)f(x,y)f(x,y) 的弯曲细线，总质量就是 fff 沿弧长的积分。

计算方法：把曲线参数化 x=x(t),y=y(t)x=x(t), y=y(t)x=x(t),y=y(t)，ds=[x′(t)]2+[y′(t)]2 dtds = \sqrt{[x'(t)]^2+[y'(t)]^2}\,dtds=[x′(t)]2+[y′(t)]2​dt。

第二类曲线积分（对坐标）

∫LP dx+Q dy\int_L P\,dx + Q\,dy∫L​Pdx+Qdy

物理意义：变力 F=(P,Q)\mathbf{F}=(P,Q)F=(P,Q) 沿曲线 LLL 做功。

计算方向必须指定——沿 LLL 正向（AAA 到 BBB）和反向（BBB 到 AAA），积分值差一个负号。

格林公式——把”线圈积分”变”面积积分”

设 LLL 是闭曲线（绕一圈回到起点），围成区域 DDD。格林公式说：

∮LP dx+Q dy=∬D(∂Q∂x−∂P∂y)dx dy\oint_L P\,dx + Q\,dy = \iint_D\left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dx\,dy∮L​Pdx+Qdy=∬D​(∂x∂Q​−∂y∂P​)dxdy

左边是曲线积分（沿边界），右边是二重积分（在内部）。边界上的信息可以从内部算出来。

这是高斯公式和斯托克斯公式的二维版本——后面两个是它的三维推广。

方向注意：LLL 取正向（沿着 LLL 走，区域 DDD 始终在左手边）。

曲面积分

第一类（对面积）：∬Σf(x,y,z) dS\iint_\Sigma f(x,y,z)\,dS∬Σ​f(x,y,z)dS。物理意义：曲面的质量。

第二类（对坐标）：∬ΣP dy dz+Q dz dx+R dx dy\iint_\Sigma P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy∬Σ​Pdydz+Qdzdx+Rdxdy。物理意义：流体流经曲面的通量（flux）。

高斯公式（散度定理）

闭曲面 Σ\SigmaΣ 围成区域 Ω\OmegaΩ：

∯ΣP dy dz+Q dz dx+R dx dy=∭Ω(∂P∂x+∂Q∂y+∂R∂z)dx dy dz\oiint_\Sigma P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy = \iiint_\Omega\left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right)dx\,dy\,dz∬​Σ​Pdydz+Qdzdx+Rdxdy=∭Ω​(∂x∂P​+∂y∂Q​+∂z∂R​)dxdydz

括号里的就是散度 ∇⋅F\nabla\cdot\mathbf{F}∇⋅F——“从该点流出多少”。高斯公式说：穿过闭曲面的总流量 = 曲面内部所有源的总和。

斯托克斯公式

空间闭曲线 LLL 和以它为边界的曲面 Σ\SigmaΣ：

∮LP dx+Q dy+R dz=∬Σ(∂R∂y−∂Q∂z)dy dz+(∂P∂z−∂R∂x)dz dx+(∂Q∂x−∂P∂y)dx dy\oint_L P\,dx + Q\,dy + R\,dz = \iint_\Sigma\left(\frac{\partial R}{\partial y}-\frac{\partial Q}{\partial z}\right)dy\,dz + \left(\frac{\partial P}{\partial z}-\frac{\partial R}{\partial x}\right)dz\,dx + \left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)dx\,dy∮L​Pdx+Qdy+Rdz=∬Σ​(∂y∂R​−∂z∂Q​)dydz+(∂z∂P​−∂x∂R​)dzdx+(∂x∂Q​−∂y∂P​)dxdy

这是旋度 ∇×F\nabla\times\mathbf{F}∇×F 的曲面积分。斯托克斯公式说：沿边界的环量 = 内部旋度的通量。

三大公式关系一览

公式 维度 把什么变什么 核心量
格林 2D 闭曲线积分 → 面积分 旋度(二维)
高斯 3D 闭曲面积分 → 体积分 散度

牛顿-莱布尼茨公式、格林公式、高斯公式、斯托克斯公式——这四个其实是同一个思想在不同维度上的表现：边界上的信息 = 内部某种”密度”的积分。数学上统称广义斯托克斯定理。

  • 旋度：四旋翼旋翼尾迹的涡量场——旋度不为零的区域就是有涡的地方

五、无穷级数——函数可以用无穷项之和来表达

概念直觉

泰勒公式说：函数 ≈ 多项式。但如果加足够多项——甚至无穷多项——能不能精确等于原函数？

  • FFT 振动分析：四旋翼飞控的 IMU 数据进 FFT（快速傅里叶变换），频谱图显示哪个频率振动超标——就是傅里叶级数的工程实现
