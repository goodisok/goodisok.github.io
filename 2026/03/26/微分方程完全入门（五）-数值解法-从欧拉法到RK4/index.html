|浏览量:

本文是《微分方程完全入门》系列的第五篇，覆盖ODE数值解法的基本方法：欧拉法、改进欧拉法、四阶龙格-库塔法（RK4），以及自适应步长和刚性方程的概念。阅读本文前需理解一阶ODE的基本形式。本文重点在方法的几何直觉和误差分析，而非纯数值分析视角。

第五篇：数值解法——从欧拉法到 RK4

一、概念引入——为什么要数值解法

在之前四篇中，我们学习了多种解析解法——分离变量、积分因子、特征根法、拉普拉斯变换。

但这些方法都有一个致命的限制：大多数现实中的ODE没有解析解。

考虑一个稍微修改的摆方程——加入非线性项和非正弦驱动力：

d2θdt2+gLsin⁡θ=F0cos⁡(ωt) \frac{d^2\theta}{dt^2} + \frac{g}{L}\sin\theta = F_0\cos(\omega t) dt2d2θ​+Lg​sinθ=F0​cos(ωt)

这个方程——受迫阻尼摆——无法用初等函数表示其解。但它描述的就是真实世界的物理。

数值解法就是回答这个问题的：用计算机逐”小步”逼近解。

二、欧拉法——最直观的起点

2.1 思想

一个一阶 ODE：dydt=f(t,y)\frac{dy}{dt} = f(t, y)dtdy​=f(t,y)，给定初始条件 y(t0)=y0y(t_0) = y_0y(t0​)=y0​。

在 (t0,y0)(t_0, y_0)(t0​,y0​) 处，切线的斜率是 f(t0,y0)f(t_0, y_0)f(t0​,y0​)。如果我们向前走一小步 Δt\Delta tΔt：

y1=y0+f(t0,y0)⋅Δt y_1 = y_0 + f(t_0, y_0) \cdot \Delta t y1​=y0​+f(t0​,y0​)⋅Δt

这就是欧拉法。

2.2 公式

yn+1=yn+f(tn,yn)⋅h y_{n+1} = y_n + f(t_n, y_n) \cdot h yn+1​=yn​+f(tn​,yn​)⋅h

其中 h=Δth = \Delta th=Δt 是步长。

三、RK4——工程界的标准

3.1 思想

欧拉法只用了区间起点处的斜率，误差 O(h)O(h)O(h)。如果能在区间内多取几个点计算斜率，然后加权平均，精度会大幅提升。

RK4 在每一步内计算 4 个斜率：

k1=f(tn,yn) k_1 = f(t_n, y_n) k1​=f(tn​,yn​) k2=f(tn+h2,yn+h2k1) k_2 = f(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_1) k2​=f(tn​+2h​,yn​+2h​k1​) k3=f(tn+h2,yn+h2k2) k_3 = f(t_n + \frac{h}{2}, y_n + \frac{h}{2}k_2) k3​=f(tn​+2h​,yn​+2h​k2​) k4=f(tn+h,yn+hk3) k_4 = f(t_n + h, y_n + h k_3) k4​=f(tn​+h,yn​+hk3​)

最终更新：

yn+1=yn+h6(k1+2k2+2k3+k4) y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4) yn+1​=yn​+6h​(k1​+2k2​+2k3​+k4​)

3.3 RK4的几何直觉

  • k1k_1k1​：起点处切线斜率（欧拉法的值）
  • k2k_2k2​：用 k1k_1k1​ 走到中点后的斜率
  • k3k_3k3​：用 k2k_2k2​ 再算一次中点斜率（修正值）
  • k4k_4k4​：用 k3k_3k3​ 走到终点后的斜率
  • 最后加权平均：(k1+2k2+2k3+k4)/6(k_1 + 2k_2 + 2k_3 + k_4)/6(k1​+2k2​+2k3​+k4​)/6

之所以 k2k_2k2​ 和 k3k_3k3​ 的权重是 2（比 k1,k4k_1, k_4k1​,k4​ 大），是因为它们的精度更好——中点的斜率最能代表整个区间的”平均斜率”。

四、数值稳定性和步长选择

4.1 稳定性

如果步长过大，欧拉法可能完全发散。

考虑 y′=−100yy' = -100yy′=−100y，y(0)=1y(0) = 1y(0)=1。如果用 h=0.02h=0.02h=0.02：

y1=1+0.02×(−100)=1−2=−1 y_1 = 1 + 0.02 \times (-100) = 1 - 2 = -1 y1​=1+0.02×(−100)=1−2=−1

一步就跳到了负值！然后继续发散。

稳定性条件：对于 y′=λyy' = \lambda yy′=λy（λ<0\lambda < 0λ<0），欧拉法要求 h<2/∣λ∣h < 2/|\lambda|h<2/∣λ∣。

RK4 的稳定性区域更大——最大步长可达约 2.78/∣λ∣2.78/|\lambda|2.78/∣λ∣。

4.2 自适应步长

现代 ODE 求解器使用自适应步长——根据局部误差估计自动调节步长。

常用方法：同时计算 RK4 和 RK5，用两者的差值估计误差；如果误差太大，步长减半；误差太小，步长加倍。

MATLAB 的 ode45 使用的就是这种”RK4(5)”方法。

4.3 刚性方程

如果系统中同时存在”很快”和”很慢”的动态：

y′=[−100000−1]y y' = \begin{bmatrix} -1000 & 0 \\ 0 & -1 \end{bmatrix} y y′=[−10000​0−1​]y

显式方法必须用小步长满足快速分量的稳定性要求——即使它早已衰减到几乎为零。

刚性方程需要隐式方法（如隐式欧拉法），它的稳定性不受快速分量限制。

在飞行动力学中，六自由度系统的快模态（荷兰滚、短周期）和慢模态（螺旋模态）并存的场景就是典型的刚性方程。

五、在 PX4 仿真中的应用

PX4 SITL 仿真中使用的是 RK4 求解器，步长通常为 1 ms 或 4 ms（与飞控主频匹配）。
