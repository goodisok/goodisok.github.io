复变函数完全入门（一）：从实数到复数——新代数结构的诞生

发表于2026-03-28|更新于2026-03-28|数学基础
|浏览量:

本文是《复变函数完全入门》系列的第一篇，覆盖复数的定义、运算、几何表示、欧拉公式和复平面。阅读本文仅需掌握高中数学中的实数运算和三角函数。本系列的目标是为理解势流理论中的保角变换打下完整的数学基础——保角变换是连接复变函数和空气动力学势流理论的数学桥梁。

第一部分：为什么要有复数

一、概念引入——一个简单方程引发的”数字危机”

1.1 问题

解方程 x2+1=0x^2 + 1 = 0x2+1=0。

实数范围内无解——因为任何实数的平方都非负。但如果我们”强行”定义一个数 iii，使得 i2=−1i^2 = -1i2=−1，那么方程的解就是 x=±ix = \pm ix=±i。

这个看起来很简单的”强行定义”，在16世纪引起了数学界激烈的争论。当时的数学家将 iii 称为”虚数”（imaginary number）——这个名称本身就带有贬义。

1.2 历史的五分钟

时间 人物 贡献
1545 卡尔达诺（Cardano） 在《大术》中首次使用 −15\sqrt{-15}−15​ 计算三次方程的解——“虚数”的首次出场
1572 邦贝利（Bombelli） 系统地使用了 iii 的运算规则，被公认为复数的先驱
1748 欧拉 给出 eiθ=cos⁡θ+isin⁡θe^{i\theta} = \cos\theta + i\sin\thetaeiθ=cosθ+isinθ——数学中最美的公式
1806 阿尔冈（Argand） 提出复平面（Argand图）——把复数”可视化”了
1832 高斯 正式采用”复数”（complex number）这个术语，并完成了复数理论的公理化

1.3 定义

复数（Complex Number） 是形如 z=a+biz = a + biz=a+bi 的数，其中 a,ba, ba,b 是实数，i=−1i = \sqrt{-1}i=−1​。

  • a=Re(z)a = \text{Re}(z)a=Re(z)——**实部**
  • b=Im(z)b = \text{Im}(z)b=Im(z)——**虚部**
  • 复数的全体记为 C\mathbb{C}C

两个复数相等：a+bi=c+dia+bi = c+dia+bi=c+di 当且仅当 a=ca=ca=c 且 b=db=db=d。

三、复数的几何表示——复平面

3.1 概念

在复平面上：

  • 水平轴 = 实轴（Re\text{Re}Re）
  • 垂直轴 = 虚轴（Im\text{Im}Im）
  • 每个复数 z=a+biz = a+biz=a+bi 对应一个点 (a,b)(a,b)(a,b)

3.2 极坐标表示

复数也可以用极坐标表示：

z=r(cos⁡θ+isin⁡θ) z = r(\cos\theta + i\sin\theta) z=r(cosθ+isinθ)

其中：

  • r=∣z∣r = |z|r=∣z∣——模长
  • θ=arg⁡(z)\theta = \arg(z)θ=arg(z)——**幅角**（argument），从正实轴开始逆时针的角度

实部和虚部的极坐标表达：

a=rcos⁡θ,b=rsin⁡θ a = r\cos\theta, \quad b = r\sin\theta a=rcosθ,b=rsinθ

3.3 手算例子

复数 z=1+iz = 1 + iz=1+i：

  • 模：∣z∣=12+12=2|z| = \sqrt{1^2 + 1^2} = \sqrt{2}∣z∣=12+12​=2​
  • 幅角：θ=arctan⁡(1/1)=π/4=45∘\theta = \arctan(1/1) = \pi/4 = 45^\circθ=arctan(1/1)=π/4=45∘

极坐标：z=2(cos⁡π4+isin⁡π4)z = \sqrt{2}(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4})z=2​(cos4π​+isin4π​)

四、欧拉公式——数学中最美的方程

4.1 公式

欧拉公式：

eiθ=cos⁡θ+isin⁡θ \boxed{e^{i\theta} = \cos\theta + i\sin\theta} eiθ=cosθ+isinθ​

这是数学中最重要的公式之一。它将三个看起来毫不相关的数学对象——指数函数、三角函数、虚数单位——联系在了一起。

4.2 特殊取值

令 θ=π\theta = \piθ=π：

eiπ+1=0 e^{i\pi} + 1 = 0 eiπ+1=0

这就是欧拉恒等式——数学中五个最重要的常数出现在同一个方程中：e,i,π,1,0e, i, \pi, 1, 0e,i,π,1,0。

4.3 极坐标的指数形式

利用欧拉公式，复数的极坐标形式可以写成：

z=reiθ \boxed{z = r e^{i\theta}} z=reiθ​

乘法的几何意义：

z1z2=r1eiθ1⋅r2eiθ2=r1r2ei(θ1+θ2) z_1 z_2 = r_1 e^{i\theta_1} \cdot r_2 e^{i\theta_2} = r_1 r_2 e^{i(\theta_1 + \theta_2)} z1​z2​=r1​eiθ1​⋅r2​eiθ2​=r1​r2​ei(θ1​+θ2​)

即：模长相乘，幅角相加。

除法的几何意义：

z1z2=r1r2ei(θ1−θ2) \frac{z_1}{z_2} = \frac{r_1}{r_2} e^{i(\theta_1 - \theta_2)} z2​z1​​=r2​r1​​ei(θ1​−θ2​)

n次幂（De Moivre公式）：

zn=rneinθ=rn(cos⁡nθ+isin⁡nθ) z^n = r^n e^{in\theta} = r^n(\cos n\theta + i\sin n\theta) zn=rneinθ=rn(cosnθ+isinnθ)

4.4 手算例子：复数乘法

z1=2eiπ/3z_1 = 2e^{i\pi/3}z1​=2eiπ/3，z2=3eiπ/6z_2 = 3e^{i\pi/6}z2​=3eiπ/6

乘积：

z1z2=2⋅3⋅ei(π/3+π/6)=6eiπ/2=6i z_1 z_2 = 2\cdot 3 \cdot e^{i(\pi/3 + \pi/6)} = 6e^{i\pi/2} = 6i z1​z2​=2⋅3⋅ei(π/3+π/6)=6eiπ/2=6i

验证（用代数形式）：

z1=2(cos⁡60∘+isin⁡60∘)=1+3iz_1 = 2(\cos 60^\circ + i\sin 60^\circ) = 1 + \sqrt{3}iz1​=2(cos60∘+isin60∘)=1+3​i z2=3(cos⁡30∘+isin⁡30∘)=332+32iz_2 = 3(\cos 30^\circ + i\sin 30^\circ) = \frac{3\sqrt{3}}{2} + \frac{3}{2}iz2​=3(cos30∘+isin30∘)=233​​+23​i

乘积：

(1+3i)(332+32i)=332+32i+92i+332i2 (1+\sqrt{3}i)(\frac{3\sqrt{3}}{2} + \frac{3}{2}i) = \frac{3\sqrt{3}}{2} + \frac{3}{2}i + \frac{9}{2}i + \frac{3\sqrt{3}}{2}i^2 (1+3​i)(233​​+23​i)=233​​+23​i+29​i+233​​i2 =332+6i−332=6i = \frac{3\sqrt{3}}{2} + 6i - \frac{3\sqrt{3}}{2} = 6i =233​​+6i−233​​=6i

✓ 一致！

五、复数在微分方程中的出现——复特征根的意义

二阶 ODE 的特征方程可能有共轭复根 r=α±iβr = \alpha \pm i\betar=α±iβ。

对应的解：

y=eαt(C1cos⁡βt+C2sin⁡βt) y = e^{\alpha t}(C_1\cos\beta t + C_2\sin\beta t) y=eαt(C1​cosβt+C2​sinβt)

复数的几何在这里变得非常自然：

  • eαte^{\alpha t}eαt——模长变化（衰减或增长）
  • eiβte^{i\beta t}eiβt——单位圆上的旋转（振荡）

整个二阶系统的动态行为，就是复数在复平面上轨迹的实数投影。
