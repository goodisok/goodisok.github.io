微分方程完全入门（一）：从变化率说起——一阶常微分方程与工程建模

本文是《微分方程完全入门》系列的第一篇，覆盖一阶常微分方程（ODE）的基本概念和求解方法。阅读本文仅需掌握一元微积分（导数、基本积分）。

第一部分：什么是微分方程

一、概念引入——一个数值计算问题的启示

1.1 问题

假设你有一架四旋翼无人机在悬停，关掉油门让它自由下落。忽略空气阻力，只受重力g=9.8 m/s2g=9.8\ \text{m/s}^2g=9.8 m/s2的作用。

它的速度 v(t)v(t)v(t) 随时间变化吗？当然。每一秒它都加速 9.8 m/s9.8\ \text{m/s}9.8 m/s。用微积分的语言：

dvdt=g \frac{dv}{dt} = g dtdv​=g

这个方程中包含了未知函数 v(t)v(t)v(t) 及其导数 dvdt\frac{dv}{dt}dtdv​。它描述的不是某个特定的速度值，而是速度如何随时间变化。

这就是微分方程。

1.2 定义

微分方程（Differential Equation） 是包含未知函数及其导数的方程。

根据自变量的个数分为两类：

  • 常微分方程（ODE）：未知函数只有一个自变量（通常为时间 ttt 或空间 xxx）
  • 偏微分方程（PDE）：未知函数有多个自变量（如温度 T(x,y,z,t)T(x,y,z,t)T(x,y,z,t)）

阶数（Order）：方程中最高阶导数的阶数

dvdt=g——一阶 ODE \frac{dv}{dt} = g \quad \text{——一阶 ODE} dtdv​=g——一阶 ODE d2xdt2+ω2x=0——二阶 ODE \frac{d^2x}{dt^2} + \omega^2 x = 0 \quad \text{——二阶 ODE} dt2d2x​+ω2x=0——二阶 ODE ∂T∂t=α∂2T∂x2——二阶 PDE（热方程） \frac{\partial T}{\partial t} = \alpha \frac{\partial^2 T}{\partial x^2} \quad \text{——二阶 PDE（热方程）} ∂t∂T​=α∂x2∂2T​——二阶 PDE（热方程）

二、一阶 ODE 的几何含义——方向场

2.1 概念引入

一个一阶 ODE 可以写成一般形式：

dydx=f(x,y) \frac{dy}{dx} = f(x, y) dxdy​=f(x,y)

它的几何含义非常直观：在平面上的每一点 (x,y)(x,y)(x,y)，我们都可以计算出一个斜率 dydx\frac{dy}{dx}dxdy​。如果把每个点关联一个短线段（箭头），就形成了方向场（slope field / direction field）。

微分方程的解就是一条处处与该方向场相切的曲线。

2.2 手算例子

考虑最简单的 ODE：

dydx=−y \frac{dy}{dx} = -y dxdy​=−y

方向场：

  • 在 y=1y=1y=1 处，斜率为 −1-1−1（下坡）
  • 在 y=2y=2y=2 处，斜率为 −2-2−2（更陡的下坡）
  • 在 y=−1y=-1y=−1 处，斜率为 111（上坡）
  • 在 y=0y=0y=0 处，斜率为 000（水平）

通解：y=Ce−xy = Ce^{-x}y=Ce−x（指数衰减）

无论从哪个点出发，解曲线都趋向于 y=0y=0y=0。这就是平衡点的概念——稍后在第四篇中讲相平面时会详细讨论。

三、一阶 ODE 的三种基本解法

3.1 可分离变量法——最简单的类型

形式：

dydx=g(x)h(y) \frac{dy}{dx} = g(x)h(y) dxdy​=g(x)h(y)

即等号右边是 xxx 的函数乘以 yyy 的函数（可因子分解）。

解法：

dyh(y)=g(x) dx \frac{dy}{h(y)} = g(x)\,dx h(y)dy​=g(x)dx

两边分别积分：

∫dyh(y)=∫g(x) dx+C \int \frac{dy}{h(y)} = \int g(x)\,dx + C ∫h(y)dy​=∫g(x)dx+C

手算例子 1：RC 电路放电

一个 RC 电路中的电压满足：

dVdt=−1RCV \frac{dV}{dt} = -\frac{1}{RC}V dtdV​=−RC1​V

其中 RRR 是电阻，CCC 是电容，RCRCRC 是时间常数。

给定初始电压 V(0)=V0V(0) = V_0V(0)=V0​。

解：

这是可分离变量的：

dVV=−1RCdt \frac{dV}{V} = -\frac{1}{RC}dt VdV​=−RC1​dt

两边积分：

∫dVV=∫−1RCdt \int \frac{dV}{V} = \int -\frac{1}{RC}dt ∫VdV​=∫−RC1​dt ln⁡∣V∣=−tRC+C1 \ln|V| = -\frac{t}{RC} + C_1 ln∣V∣=−RCt​+C1​ V=e−t/RC+C1=Ce−t/RC V = e^{-t/RC + C_1} = Ce^{-t/RC} V=e−t/RC+C1​=Ce−t/RC

代入初始条件 V(0)=V0V(0) = V_0V(0)=V0​：

V0=C⋅e0=C V_0 = C \cdot e^0 = C V0​=C⋅e0=C

特解：

V(t)=V0e−t/RC \boxed{V(t) = V_0 e^{-t/RC}} V(t)=V0​e−t/RC​

物理意义：电压按指数衰减到 0。RCRCRC 越大，衰减越慢。在飞控电路中，电容滤波的原理正是这个方程——时间常数决定了滤波器的截止频率。

手算例子 2：自由落体（带空气阻力）

更真实的情况是考虑空气阻力——阻力与速度成正比：

mdvdt=mg−kv m\frac{dv}{dt} = mg - kv mdtdv​=mg−kv

即：

dvdt=g−kmv \frac{dv}{dt} = g - \frac{k}{m}v dtdv​=g−mk​v

解：

分离变量：

dvg−(k/m)v=dt \frac{dv}{g - (k/m)v} = dt g−(k/m)vdv​=dt

令 a=ga = ga=g，b=k/mb = k/mb=k/m：

∫dva−bv=∫dt \int \frac{dv}{a - bv} = \int dt ∫a−bvdv​=∫dt −1bln⁡∣a−bv∣=t+C -\frac{1}{b}\ln|a - bv| = t + C −b1​ln∣a−bv∣=t+C

代入初始条件 v(0)=0v(0) = 0v(0)=0，得到 C=−1bln⁡aC = -\frac{1}{b}\ln aC=−b1​lna，整理：

v(t)=mgk(1−e−(k/m)t) \boxed{v(t) = \frac{mg}{k}\left(1 - e^{-(k/m)t}\right)} v(t)=kmg​(1−e−(k/m)t)​

物理意义：速度不会无限增大，而是趋向于终端速度（terminal velocity） vT=mg/kv_T = mg/kvT​=mg/k。这在四旋翼的下降速率限制、或者截击机的速度边界中都有体现。

3.2 一阶线性 ODE——积分因子法

标准形式：

dydx+P(x)y=Q(x) \frac{dy}{dx} + P(x)y = Q(x) dxdy​+P(x)y=Q(x)

如果 Q(x)=0Q(x)=0Q(x)=0，是齐次方程；Q(x)≠0Q(x) \neq 0Q(x)=0，是非齐次方程。

解法——积分因子法：

  1. 计算积分因子 μ(x)=e∫P(x) dx\mu(x) = e^{\int P(x)\,dx}μ(x)=e∫P(x)dx
  2. 两边乘以 μ(x)\mu(x)μ(x)：μ(x)dydx+μ(x)P(x)y=μ(x)Q(x) \mu(x)\frac{dy}{dx} + \mu(x)P(x)y = \mu(x)Q(x) μ(x)dxdy​+μ(x)P(x)y=μ(x)Q(x)
  3. 左边变成 (μy)′(\mu y)'(μy)′：ddx(μy)=μQ \frac{d}{dx}(\mu y) = \mu Q dxd​(μy)=μQ
  4. 两边积分：μy=∫μQ dx+C \mu y = \int \mu Q\,dx + C μy=∫μQdx+C
  5. 解出 yyy。

手算例子：RL 电路

一个 RL 电路（电阻+电感串联，施加电压 EEE）：

Ldidt+Ri=E L\frac{di}{dt} + Ri = E Ldtdi​+Ri=E

标准形式：

didt+RLi=EL \frac{di}{dt} + \frac{R}{L}i = \frac{E}{L} dtdi​+LR​i=LE​

解：

P(t)=R/LP(t) = R/LP(t)=R/L，μ(t)=e∫R/L dt=eRt/L\mu(t) = e^{\int R/L\,dt} = e^{Rt/L}μ(t)=e∫R/Ldt=eRt/L

两边乘 μ\muμ：

eRt/Ldidt+RLeRt/Li=ELeRt/L e^{Rt/L}\frac{di}{dt} + \frac{R}{L}e^{Rt/L}i = \frac{E}{L}e^{Rt/L} eRt/Ldtdi​+LR​eRt/Li=LE​eRt/L

即：

ddt(ieRt/L)=ELeRt/L \frac{d}{dt}\left(i e^{Rt/L}\right) = \frac{E}{L}e^{Rt/L} dtd​(ieRt/L)=LE​eRt/L

积分：

ieRt/L=EL∫eRt/Ldt=EL⋅LReRt/L+C=EReRt/L+C i e^{Rt/L} = \frac{E}{L}\int e^{Rt/L}dt = \frac{E}{L}\cdot\frac{L}{R}e^{Rt/L} + C = \frac{E}{R}e^{Rt/L} + C ieRt/L=LE​∫eRt/Ldt=LE​⋅RL​eRt/L+C=RE​eRt/L+C

所以：

i(t)=ER+Ce−Rt/L i(t) = \frac{E}{R} + Ce^{-Rt/L} i(t)=RE​+Ce−Rt/L

代入初始条件 i(0)=0i(0) = 0i(0)=0 得 C=−E/RC = -E/RC=−E/R：

i(t)=ER(1−e−(R/L)t) \boxed{i(t) = \frac{E}{R}\left(1 - e^{-(R/L)t}\right)} i(t)=RE​(1−e−(R/L)t)​

物理意义：电流以时间常数 τ=L/R\tau = L/Rτ=L/R 趋近稳态值 E/RE/RE/R。在无人机电机驱动器中，电机的电学模型正是 RL 电路——这和 BLDC 电机的电流响应直接相关。

3.3 恰当微分方程

形式：

M(x,y) dx+N(x,y) dy=0 M(x,y)\,dx + N(x,y)\,dy = 0 M(x,y)dx+N(x,y)dy=0

恰条件：

∂M∂y=∂N∂x \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} ∂y∂M​=∂x∂N​

如果条件满足，存在一个函数 ψ(x,y)\psi(x,y)ψ(x,y) 使得 dψ=Mdx+Ndyd\psi = Mdx + Ndydψ=Mdx+Ndy，通解为 ψ(x,y)=C\psi(x,y) = Cψ(x,y)=C。

找 ψ\psiψ 的方法：从 ∂ψ∂x=M\frac{\partial \psi}{\partial x} = M∂x∂ψ​=M 积分得到 ψ=∫Mdx+g(y)\psi = \int M dx + g(y)ψ=∫Mdx+g(y)，再用 ∂ψ∂y=N\frac{\partial \psi}{\partial y} = N∂y∂ψ​=N 确定 g(y)g(y)g(y)。

手算例子

检验并求解：

(2xy+3) dx+(x2−1) dy=0 (2xy + 3)\,dx + (x^2 - 1)\,dy = 0 (2xy+3)dx+(x2−1)dy=0

检验恰条件：

∂M∂y=2x,∂N∂x=2x \frac{\partial M}{\partial y} = 2x, \quad \frac{\partial N}{\partial x} = 2x ∂y∂M​=2x,∂x∂N​=2x

相等，恰当！

求 ψ\psiψ：

ψ=∫(2xy+3) dx=x2y+3x+g(y) \psi = \int (2xy + 3)\,dx = x^2y + 3x + g(y) ψ=∫(2xy+3)dx=x2y+3x+g(y) ∂ψ∂y=x2+g′(y)=x2−1 \frac{\partial \psi}{\partial y} = x^2 + g'(y) = x^2 - 1 ∂y∂ψ​=x2+g′(y)=x2−1

所以 g′(y)=−1g'(y) = -1g′(y)=−1，g(y)=−y+C1g(y) = -y + C_1g(y)=−y+C1​

因此：

ψ(x,y)=x2y+3x−y=C \psi(x,y) = x^2y + 3x - y = C ψ(x,y)=x2y+3x−y=C

这就是通解。

五、一阶 ODE 在飞控与仿真中的应用

5.1 一阶控制系统

PID 控制器中的 PI 环节本质上是一阶 ODE。一个比例-积分控制器：

u(t)=Kpe(t)+Ki∫0te(τ) dτ u(t) = K_p e(t) + K_i \int_0^t e(\tau)\,d\tau u(t)=Kp​e(t)+Ki​∫0t​e(τ)dτ

写成微分方程形式：

dudt=Kpdedt+Kie(t) \frac{du}{dt} = K_p\frac{de}{dt} + K_i e(t) dtdu​=Kp​dtde​+Ki​e(t)

5.2 气压计高度滤波

无人机气压计测量高度时使用了低通滤波——本质上就是求解一阶ODE：

dhfiltereddt=1τ(hraw−hfiltered) \frac{dh_{\text{filtered}}}{dt} = \frac{1}{\tau}(h_{\text{raw}} - h_{\text{filtered}}) dtdhfiltered​​=τ1​(hraw​−hfiltered​)

这是一个一阶线性ODE，解就是指数趋近真实高度。τ\tauτ 越大，滤波越平滑但延迟越大。

5.3 电机响应

四旋翼电机的转速响应近似为一阶系统：

dωdt=−1τω+Kτu \frac{d\omega}{dt} = -\frac{1}{\tau}\omega + \frac{K}{\tau}u dtdω​=−τ1​ω+τK​u

其中 uuu 是 PWM 输入，τ\tauτ 是电机时间常数（通常 50-200 ms）。这个模型直接决定了飞控的响应速度。
