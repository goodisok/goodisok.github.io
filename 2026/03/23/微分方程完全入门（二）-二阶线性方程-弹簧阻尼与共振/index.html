微分方程完全入门（二）：二阶线性方程——弹簧、阻尼与共振

|浏览量:

本文是《微分方程完全入门》系列的第二篇，覆盖二阶线性常微分方程的完整理论。阅读本文前需熟悉一阶ODE的解法。掌握本篇后你将理解四旋翼飞行动力学的”阻尼比”和”自然频率”这两个核心参数。

第二篇：二阶线性方程——弹簧、阻尼与共振

一、二阶 ODE 从哪来——三个物理问题

1.1 概念引入

我们身边的很多物理系统都可以用一个统一的方程来描述：

弹簧-质量-阻尼系统：

md2xdt2+cdxdt+kx=F(t) m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = F(t) mdt2d2x​+cdtdx​+kx=F(t)

RLC电路：

Ld2qdt2+Rdqdt+1Cq=E(t) L\frac{d^2q}{dt^2} + R\frac{dq}{dt} + \frac{1}{C}q = E(t) Ldt2d2q​+Rdtdq​+C1​q=E(t)

四旋翼俯仰角动力学（简化）：

Id2θdt2+cθdθdt+kθθ=τ(t) I\frac{d^2\theta}{dt^2} + c_\theta\frac{d\theta}{dt} + k_\theta\theta = \tau(t) Idt2d2θ​+cθ​dtdθ​+kθ​θ=τ(t)

三个系统，同一个数学结构——二阶线性常系数ODE。

1.2 定义

标准形式：

ad2ydt2+bdydt+cy=f(t) a\frac{d^2y}{dt^2} + b\frac{dy}{dt} + cy = f(t) adt2d2y​+bdtdy​+cy=f(t)
  • a,b,ca, b, ca,b,c 是常数
  • f(t)f(t)f(t) 是源项（外力、输入电压、控制力矩）
  • 如果 f(t)=0f(t)=0f(t)=0，称为齐次方程；否则称为非齐次方程

二、齐次解——特征根法

2.1 核心思想

对于齐次方程 ay′′+by′+cy=0ay'' + by' + cy = 0ay′′+by′+cy=0，我们假设解的形式为 y=erty = e^{rt}y=ert。

代入：

ar2ert+brert+cert=0 a r^2 e^{rt} + b r e^{rt} + c e^{rt} = 0 ar2ert+brert+cert=0

消去 ert≠0e^{rt} \neq 0ert=0，得到特征方程：

ar2+br+c=0 \boxed{a r^2 + b r + c = 0} ar2+br+c=0​

这是一个代数方程——我们把微分方程问题转化成了二次方程求根问题。

2.2 三种情况

特征根 r=−b±b2−4ac2ar = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}r=2a−b±b2−4ac​​ 有三种情况：

情况 1：两个不等实根（b2−4ac>0b^2 - 4ac > 0b2−4ac>0，过阻尼）

y=C1er1t+C2er2t y = C_1 e^{r_1 t} + C_2 e^{r_2 t} y=C1​er1​t+C2​er2​t

例：y′′+3y′+2y=0y'' + 3y' + 2y = 0y′′+3y′+2y=0

特征方程：r2+3r+2=0⇒(r+1)(r+2)=0⇒r1=−1,r2=−2r^2 + 3r + 2 = 0 \Rightarrow (r+1)(r+2)=0 \Rightarrow r_1=-1, r_2=-2r2+3r+2=0⇒(r+1)(r+2)=0⇒r1​=−1,r2​=−2

通解：y=C1e−t+C2e−2ty = C_1 e^{-t} + C_2 e^{-2t}y=C1​e−t+C2​e−2t

两个指数都是衰减的。系统回到平衡但没有振荡。

情况 2：重根（b2−4ac=0b^2 - 4ac = 0b2−4ac=0，临界阻尼）

y=(C1+C2t)ert y = (C_1 + C_2 t) e^{r t} y=(C1​+C2​t)ert

例：y′′+4y′+4y=0y'' + 4y' + 4y = 0y′′+4y′+4y=0

特征方程：r2+4r+4=0⇒(r+2)2=0⇒r=−2r^2 + 4r + 4 = 0 \Rightarrow (r+2)^2 = 0 \Rightarrow r = -2r2+4r+4=0⇒(r+2)2=0⇒r=−2（重根）

通解：y=(C1+C2t)e−2ty = (C_1 + C_2 t)e^{-2t}y=(C1​+C2​t)e−2t

关键性质：由于 tertt e^{rt}tert 项的存在，解先上升后衰减。这是”最快回到平衡但不超调”的情况——控制工程中的理想阻尼状态。

情况 3：共轭复根（b2−4ac<0b^2 - 4ac < 0b2−4ac<0，欠阻尼）

设 r=α±iβr = \alpha \pm i\betar=α±iβ，则：

y=eαt(C1cos⁡βt+C2sin⁡βt) y = e^{\alpha t}(C_1\cos\beta t + C_2\sin\beta t) y=eαt(C1​cosβt+C2​sinβt)

例：y′′+2y′+5y=0y'' + 2y' + 5y = 0y′′+2y′+5y=0

特征方程：r2+2r+5=0⇒r=−1±2ir^2 + 2r + 5 = 0 \Rightarrow r = -1 \pm 2ir2+2r+5=0⇒r=−1±2i

通解：y=e−t(C1cos⁡2t+C2sin⁡2t)y = e^{-t}(C_1\cos 2t + C_2\sin 2t)y=e−t(C1​cos2t+C2​sin2t)

物理含义：指数衰减的震荡——震荡幅度以 eαte^{\alpha t}eαt 的速度衰减。

三、阻尼比与自然频率——工程中最常用的两个参数

3.1 概念

二阶系统的标准工程表达为：

d2ydt2+2ζωndydt+ωn2y=0 \frac{d^2y}{dt^2} + 2\zeta\omega_n\frac{dy}{dt} + \omega_n^2 y = 0 dt2d2y​+2ζωn​dtdy​+ωn2​y=0

其中：

  • ωn\omega_nωn​——**无阻尼自然频率**（undamped natural frequency），rad/s
  • ζ\zetaζ——**阻尼比**（damping ratio），无量纲

与参数 a,b,ca,b,ca,b,c 的关系：

ωn=ca,ζ=b2ac \omega_n = \sqrt{\frac{c}{a}}, \quad \zeta = \frac{b}{2\sqrt{ac}} ωn​=ac​​,ζ=2ac​b​

3.2 三种阻尼状态的工程意义

阻尼比 状态 特征根 响应特征 工程应用
ζ<1\zeta < 1ζ<1 欠阻尼 共轭复数 震荡衰减 大多数飞行器模态
ζ=1\zeta = 1ζ=1 临界阻尼 重实根 最快回复无超调 火炮瞄准系统、理想飞控
ζ>1\zeta > 1ζ>1 过阻尼 两负实根 缓慢回复无震荡 气压计滤波、某些机械阻尼器

3.3 手算例子：四旋翼俯仰响应

一架四旋翼的俯仰角动力学近似为（忽略推力矢量耦合）：

θ¨+2ζωnθ˙+ωn2θ=Kωn2θd \ddot{\theta} + 2\zeta\omega_n\dot{\theta} + \omega_n^2\theta = K\omega_n^2\theta_d θ¨+2ζωn​θ˙+ωn2​θ=Kωn2​θd​

其中 θd\theta_dθd​ 是目标俯仰角，KKK 是增益。

以典型小型四旋翼（~1.5 kg）为例：

ωn≈30 rad/s, ζ≈0.7 \omega_n \approx 30\ \text{rad/s},\ \zeta \approx 0.7 ωn​≈30 rad/s, ζ≈0.7

特征根：

r=−ζωn±ωnζ2−1=−21±300.49−1=−21±30i0.51≈−21±21.4i r = -\zeta\omega_n \pm \omega_n\sqrt{\zeta^2-1} = -21 \pm 30\sqrt{0.49-1} = -21 \pm 30i\sqrt{0.51} \approx -21 \pm 21.4i r=−ζωn​±ωn​ζ2−1​=−21±300.49−1​=−21±30i0.51​≈−21±21.4i

解的形式：

θ(t)=e−21t(C1cos⁡21.4t+C2sin⁡21.4t)+θsteady \theta(t) = e^{-21t}(C_1\cos 21.4t + C_2\sin 21.4t) + \theta_{\text{steady}} θ(t)=e−21t(C1​cos21.4t+C2​sin21.4t)+θsteady​

物理意义：

  • 衰减时间：约 1/(ζωn)≈0.048 s1/(\zeta\omega_n) \approx 0.048\ \text{s}1/(ζωn​)≈0.048 s（48 ms）——系统调节时间的量级
  • 振荡频率：ωd=ωn1−ζ2≈21.4 rad/s\omega_d = \omega_n\sqrt{1-\zeta^2} \approx 21.4\ \text{rad/s}ωd​=ωn​1−ζ2​≈21.4 rad/s——约 3.4 Hz
  • 与PID的关系：KpK_pKp​ 主要影响 ωn\omega_nωn​（响应速度），KdK_dKd​ 主要影响 ζ\zetaζ（阻尼程度）

这个模型正是你那篇 PX4-PID调参文章 中”率环→姿态环”数值的数学基础。

四、非齐次方程——待定系数法

4.1 问题

对于非齐次方程 ay′′+by′+cy=f(t)ay'' + by' + cy = f(t)ay′′+by′+cy=f(t)，通解为：

y=yh+yp y = y_h + y_p y=yh​+yp​

4.3 手算例子：受迫振动

d2xdt2+4x=sin⁡2t \frac{d^2x}{dt^2} + 4x = \sin 2t dt2d2x​+4x=sin2t

齐次解：r2+4=0⇒r=±2i⇒xh=C1cos⁡2t+C2sin⁡2tr^2 + 4 = 0 \Rightarrow r = \pm 2i \Rightarrow x_h = C_1\cos 2t + C_2\sin 2tr2+4=0⇒r=±2i⇒xh​=C1​cos2t+C2​sin2t

注意：f(t)=sin⁡2tf(t) = \sin 2tf(t)=sin2t 正好是齐次解的一部分！所以猜测 xp=t(Asin⁡2t+Bcos⁡2t)x_p = t(A\sin 2t + B\cos 2t)xp​=t(Asin2t+Bcos2t)

代入后解得 A=0,B=−14A = 0, B = -\frac{1}{4}A=0,B=−41​：

x(t)=C1cos⁡2t+C2sin⁡2t−t4cos⁡2t x(t) = C_1\cos 2t + C_2\sin 2t - \frac{t}{4}\cos 2t x(t)=C1​cos2t+C2​sin2t−4t​cos2t

物理意义：最后一项 −t4cos⁡2t-\frac{t}{4}\cos 2t−4t​cos2t 的振幅随时间线性增长——这就是共振的数学描述。当激励频率等于系统自然频率时，即使很小的激励也能产生很大的响应。

在四旋翼中，共振意味着：如果电机旋转频率与某个机体结构模态一致，会导致剧烈的振动——这是结构设计中需要避开的情况。
